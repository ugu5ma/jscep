org.jscep.util
Class AlgorithmDictionary

java.lang.Object
  org.jscep.util.AlgorithmDictionary

public final class AlgorithmDictionary
extends Object

This class provides a utility to lookup a friendly name for an algorithm given a particular OID or AlgorithmIdentifier.

The internal dictionary is by no means comprehensive, and new algorithms are generally as and when they are required by changes to the SCEP specification.

David Grant

getTransformation

public static String getTransformation(String cipher)
                                throws IllegalArgumentException
Returns an appropriate transformation name for the given cipher.

In the JCA, an instance of a Cipher may be obtained using just the cipher algorithm (e.g. DES), but this is subject to defaults specified by the provider, so we "fill out" the cipher name in this method, so that DES becomes DES/CBC/PKC5Padding

Parameters:
cipher - the cipher algorithm name.
Returns:
the transformation name
Throws:
IllegalArgumentException - if the cipher algorithm is not appropriate.
