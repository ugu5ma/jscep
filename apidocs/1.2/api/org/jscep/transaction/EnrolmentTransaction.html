Class EnrolmentTransaction

java.lang.Object
  org.jscep.transaction.Transaction
      org.jscep.transaction.EnrolmentTransaction

public class EnrolmentTransaction
extends Transaction

This class represents a SCEP transaction, and provides a framework for performing operations.

The behaviour of this class changes in accordance with the possible valid states for each transaction operation. For enrolment operations, clients should inspect the Transaction.State returned by the send() or poll().
